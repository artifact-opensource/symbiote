Capabilities

Built different.

Not another wrapper around an API. Symbiote is a persistent daemon — messaging-first, interrupt-driven, local by default.

⚡
Real-time Interrupts
Say “stop” mid-turn and the agent stops. Immediately. Not after the current tool call finishes — now. True interrupt-driven execution.
.079;
Multi-Channel Native
Discord, WhatsApp, HTTP API — same agent, same memory, same identity. One process handles all channels simultaneously with message coalescing.
🧠
Persistent Memory
Sessions survive restarts. COMB for lossless cross-session memory, HEKTOR for sub-millisecond semantic search. Your agent remembers everything.
.095;
14+ Tools + MCP Bridge
File I/O, shell exec, web fetch, image analysis, TTS, process management, and more — built in. Connect any MCP server for unlimited extension.
🤖
Sub-Agent Orchestration
Spawn child agents for parallel tasks with depth control, independent iteration budgets, and result collection. Delegate complex work without blocking.
.074;
Local-First
Your machine, your data, your keys. No Docker required. No Redis. No cloud dependencies. CPU-only — if it runs Node.js 20+, it runs Symbiote.

Get Started

Up and running in 60 seconds.

Clone, configure, run. That’s it.

terminal 
# Clone the repo
git clone https://github.com/Artifact-Virtual/symbiote.git
cd symbiote

# Install dependencies
npm install

# Copy and edit the config
cp symbiote.example.json symbiote.json
$EDITOR symbiote.json  # add your API keys

# Launch the daemon
npm start

# Or run directly
npx ts-node symbiote-core/src/daemon.ts
